## Installation

Install my-project with npm

```bash
  npm install
  npm start
```
    
## Run Locally

Clone the project

```bash
  git clone git@github.com:aleksandarLazic1998/rtk-template.git
 
```

Go to the project directory

```bash
  cd rtk-template.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
